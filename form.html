To use this form, you need to set up a Google Apps Script web app and provide its URL.

If you deployed the server, enter its URL here for better reliability

How to get your Google Apps Script URL:

  1. Create a Google Sheet
  2. Go to Extensions → Apps Script
  3. Copy the code from google-apps-script.js
  4. Deploy as Web App (Execute as: Me, Access: Anyone)
  5. Copy the Web App URL and paste it above
